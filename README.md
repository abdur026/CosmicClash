# CosmicClash

## Description 

Welcome to Cosmic Clash! This exciting iOS app lets you pilot a rocketship to protect the galaxy from UFO invaders. Just tilt your device to steer the rocketship and fire missiles to defeat the UFOs. Some UFOs might need a few hits to go down. Plus, you can log in with your email or Google account to save your high scores and compete on the leaderboard. Enjoy the game!

## Installation

To enjoy the Rocketship Game, follow these simple steps:

1. **Download Xcode**: If you don't have Xcode already, you can download it from [here](https://developer.apple.com/xcode/).

2. **Clone the Repository:**
- Visit our [GitHub repository](https://github.com/abdur026/CosmicClash) and click the "Clone" button to get the project files.
- Open Terminal and navigate to the directory where you want to store the project.
- Run the following command to clone the repository:
```
git clone https://github.com/your-username/rocketship-game.git
```
3. **Open the Project:**

- Launch Xcode.

- Click on "File" -> "Open" and select the Rocketship Game project folder you just cloned.

4. **Build and Run:**
- Connect your iOS device to your Mac or use the simulator.

- Click the "Run" button in Xcode to build and run the app on your device or simulator.

## Dependencies

1. **Firebase iOS SDK:** This library enables features like leaderboard and score saving. To integrate it into your project, follow the instructions at https://github.com/firebase/firebase-ios-sdk.

2. **Google Sign-In for iOS:** This dependency allows users to log in with their Google accounts. You can add it to your project by following the steps at https://github.com/google/GoogleSignIn-iOS.git.

Remember to download and configure these dependencies before building the project.
